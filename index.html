What is PixelNunc

TL;DR :

  • It is a minted NFT Objkt Number 34102 on the HicEtNunc platform
  • It is a collaborative Pixel Canvas open for all to paint dots!
  • It is powered by an API and data on the H=N contract
  • To paint a pixel: Mint an Objkt with a properly formatted title. Burn it or send it to PixelNunc. That’s is! Your Pixel will be painted on canvas.

Go and visit the PixelNunc, read more about the project below, or see the how to paint

PixelNunc is a canvas, it is only what we make it to be.

Description

PixelNunc is a collaborative 64x64 pixel canvas, driven by data on the H=N contract. Anyone can draw, set, or replace a pixel on this canvas by minting an Objkt with a properly formatted title, example: x12y24#FF0000. Once the Objkt with that title is minted, in order to apply the pixel to the Canvas, it should be either burned, or sent to the PixelNunc’s address : tz1QLqKRozFS4xKTjmeqQ5LPp7pzmoKJ8AmP

Duration

The project duration is fluid, but will last at least 100 days. At its end, the canvas will freeze at its latest version. Hopefully a collaborative final drawing made by multiple participants will exist at that time.

In order to freeze at the last image, The NFT will remain dependent on and still call the API, but the API endpoint will be converted into a static Json file.

About the burned pixels or those sent to the pixelNunc

Since there are potentially many pixels that might be sent to the PixelNunc’s address, please note I will not be able to return the objkts to their senders at the end of the project since that might involve a lot of time and gas fees.

Project workflow and accompanying artworks

This is a fluid project. At different stages, there will be accompanying NFTs created of the PixelNunc or animations of its development.
Some early pixel contributors will recieve free objkts.

I want to paint on the PixelNunc, but does it affect my collection ?

If you want the pixels you paint to show in your artists creations you could send one token to the PixelNunc’s address and giveaway, or sell the rest for free.

If you want to paint on the PixelNunc but do not want those pixels to interfere with your collection you can mint them and then burn all tokens.
